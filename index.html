Simbolos Especiais

Vamos adiconar alguns simbolos especiais:
®
©
™
€
$

para usar estes simbolos especiais é simples, para usar o simbolo de copyright digite o código:

&copy;

para imprimir o simbolo de R, basta color o código:

&reg;

para imprimir o simbolo de TM , basta colocar o código:

&trade;

para imprimir o simbolo de euro basta colocar o código:

&euro;

para imprimir o simbolo de dollar basta colocar o código

&dollar;

Existem vários outros simbolos especias, mas esses são só alguns exemplos.

Emojis

Vamos adicionar alguns emojis:

🤓

O primeiro passo para adicionar emojis no seu documento html é acessar o site Emojipedia

Ao entrar no Emojipedia pesquise o emoji que deseja usar, vá até o fim da página e lá você vai encontrar um código, copie apenas o número do código.

Após fazer isso, entre no seu editor de HTML e digite:

&#x"o número que você copiou";
Exemplo: &#x1f913;
